## 2. 題目說明

一開始有 $N$ 瓶可樂，喝完後的空瓶可以每 3 個再換 1 瓶新的可以先借空瓶來湊數，最後喝完剩下的空瓶數量要足夠歸還
- **輸入:** 一個整數 $N$ ($1 \le N \le 200$)，代表一開始擁有的可樂瓶數
- **預期輸出:** 最多可以喝到多少瓶可樂
- **主要規則或限制**
  1. 每 3 個空瓶可以換 1 瓶新可樂
  2. 可以借空瓶來湊齊 3 個換可樂，但喝完後產生的空瓶必須歸還
- **核心任務:** 計算含初始可樂、兌換可樂在內的最高總飲用量

## 3. 思考邏輯與解題策略

### 初始想法

- 之前就有寫過此題，直接套用公式解

### 最終策略

- 數學公式法，總數為 $N + \lfloor N/2 \rfloor = \lfloor 3N/2 \rfloor$

## 4. 虛擬碼

END
